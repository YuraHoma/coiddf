# Client Preview Deploy

Static preview for the charity website redesign.

## Deploy Options

### Vercel

```bash
npx vercel deploy --prod --yes
```

Run from this folder:

```bash
cd outputs/client-deploy
```

### Netlify

```bash
npx netlify-cli deploy --prod --dir .
```

Run from this folder:

```bash
cd outputs/client-deploy
```
