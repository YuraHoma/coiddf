# Сайт БО «Фонд розвитку міжнародної співпраці» (ICDF)

Статичний сайт на [Eleventy](https://www.11ty.dev/) з редагуванням контенту через [Pages CMS](https://pagescms.org).

- **Продакшн:** https://coiddf.homchenkoyura.workers.dev/ (Cloudflare)
- **CMS:** Pages CMS — конфігурація у `.pages.yml`, редагування комітяться напряму в цей репозиторій.

## Розробка

```bash
npm install
npm run serve   # дев-сервер на http://localhost:8745
npm run build   # збірка у _site/
```

## Структура

- `src/_data/*.json` — контент сторінок (редагується через Pages CMS)
- `src/news/*.md`, `src/projects/*.md` — новини та проєкти
- `src/_includes/` — шаблони (Nunjucks)
- `src/_headers` — security-заголовки для Cloudflare
- `.pages.yml` — схема Pages CMS

## Деплой

Cloudflare Workers (конфігурація — `wrangler.jsonc`): збірка `npm run build`, роздається папка `_site`.

- **Автодеплой:** кожен push у `main` збирається й деплоїться через GitHub Actions (`.github/workflows/deploy.yml`; секрети `CLOUDFLARE_API_TOKEN` і `CLOUDFLARE_ACCOUNT_ID` в налаштуваннях репозиторію).
- **Вручну:** `npm run deploy` (потрібен вхід через `npx wrangler login` або `CLOUDFLARE_API_TOKEN`).

## Перед публічним запуском

- Замінити референсні фото на власні (оптимізація у WebP відбувається автоматично при збірці).
- Замінити намальовані логотипи партнерів на офіційні (з дозволу партнерів).
- Вказати реальні посилання на соцмережі у налаштуваннях сайту (блок у футері з'явиться автоматично).
- Додати файли документів на сторінку «Звітність» (зараз список із приміткою «буде опубліковано»).
- Затвердити текст сторінки «Політика» керівництвом і прибрати примітку-чернетку.
- За потреби англійської версії — додати EN-контент (фейковий перемикач мови прибрано).
